## Single-Page HTML CV

A simple single-page Curriculum Vitae (CV) built using only HTML. This project demonstrates semantic HTML structure, SEO best practices, Open Graph tags, and favicon integration.

## Features

- Semantic HTML5 structure
- Single-page CV layout
- SEO meta tags
- Open Graph (OG) tags
- Favicon support
- Ready for future CSS styling

## Project Structure

single-page-cv/
├── index.html
├── favicon.ico
└── README.md

## Sections Included

- Personal Information
- Professional Summary
- Skills
- Education
- Experience / Career History
- Contact Information
